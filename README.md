Мій проєкт
Цей проєкт створений для практики основ Git та GitHub. Він містить мінімальну структуру веб‑сторінки з файлів **index.html** та **style.css**, а також базове оформлення та текстовий контент.  
У процесі роботи були виконані завдання з ініціалізації Git, створення комітів, роботи з гілками, merge, створення `.gitignore` та публікації репозиторію на GitHub.

## Структура

- `index.html` — головна HTML‑сторінка
- `style.css` — стилі для сторінки
- `.gitignore` — правила для ігнорування службових та системних файлів
- `README.md` — опис проєкту

## Як відкрити проєкт

1. Завантаж або склонуй репозиторій на свій комп’ютер.
2. Відкрий папку проєкту.
3. Запусти файл **index.html** у будь‑якому браузері (Chrome, Firefox, Edge тощо).

## Виконані Git‑завдання

- Ініціалізовано локальний Git‑репозиторій.
- Створено мінімум два коміти:
  - початкова версія проєкту;
  - зміни в HTML або CSS.
- Створено окрему гілку для нових змін.
- Виконано merge гілки у `main` без конфліктів.
- Додано файл `.gitignore` та закомічено його.
- Під’єднано локальний репозиторій до GitHub.
- Код успішно завантажено у віддалений репозиторій.

## Мета роботи

- Закріпити навички роботи з Git.
- Навчитися працювати з гілками та merge.
- Навести порядок у репозиторії.
- Оформити базовий README.md для GitHub.
